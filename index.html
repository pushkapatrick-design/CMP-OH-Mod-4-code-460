Code 460 - Insuf Grnd Clr CIP Wire - NonVehic Traffic

When to Use Code 460

Insufficient pole-to-pole or pole-to-building CIP ground clearance where vehicles do not travel.

Minimum Clearances

10' for residential, 12' for commercial.

Required Documentation

Provide CIP name, measurement, and description.

Pedestrian Areas Only

Areas accessible to pedestrians only.
